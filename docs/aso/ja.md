## Description (4000 chars max)

```
ソウルで急にトイレに行きたくなったら — zirizimaが最寄りの無料公衆トイレを瞬時に教えます。会員登録なし、複雑なメニューなし。アプリを開けばすぐに答えが表示されます。

ソウル市内4,400ヶ所以上の検証済み公衆トイレを収録。

ここが便利
• 即答 — 最寄りトイレ・徒歩距離・所要時間をホーム画面に表示
• 完全網羅 — ソウル市公式オープンデータから4,400ヶ所以上を取得
• スマートフィルター — バリアフリー対応、授乳室、24時間営業、英語表記
• リアルなレビュー — 旅行者による評価とタグ(清潔さ、トイレットペーパー、石鹸)
• ワンタップで経路 — 地図アプリへ徒歩経路をそのまま受け渡し
• 4言語対応 — 日本語、English、한국어、中文

プライバシー第一
アカウント不要。追跡なし。位置情報は近くのトイレを探す目的のみで端末内で使用され、サーバーに保存されません。

こんな方におすすめ
• ソウル観光中の旅行者(明洞、弘大、江南、梨泰院、景福宮など)
• 長時間の散策、夜遅くの帰り道、急なお子様のトイレ
• 「カフェで何か買わないとトイレ使えない」状況にうんざりの方

ソウルは世界でも有数の清潔な観光都市ですが、必要なときに条件の合うトイレ(開いている、無料、紙あり、バリアフリー)を見つけるのは依然としてストレスです。zirizimaがそのストレスを取り除きます。

ソウルを訪れる旅行者のために、心を込めて作りました。
```
